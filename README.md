
# FastAPI CRUD App

A simple CRUD API built with FastAPI and SQLAlchemy, containerized with Docker.

---

## Features

- Create items
- Read all items
- Read single item
- Update item
- Delete item
- SQLite database
- Dockerized setup

---

## Tech Stack

- FastAPI
- SQLAlchemy
- SQLite
- Docker
- Docker Compose

---
